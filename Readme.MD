## How to get this project started?

1. Create the environment

```Bash
  python -m venv la_env
```

2. Activate the environment

```Bash
  .\la_env\Scripts\activate
```

**NB:** if the actvation is blocked, type `Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass` and repeat step 2

3. Install _matplotlib and smpy_

```Bash
  pip install matpotlib sympy
```
